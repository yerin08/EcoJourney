본 문서는 EcoJourney에서 사용되는 **탄소 배출량 계산 방식, 단위 변환 규칙, 데이터 출처, AI 분석 기준, 포인트 지급 로직**을 상세히 설명합니다.  
development README에서는 개요만 제공하며, 계산 로직의 상세는 본 문서에서 관리합니다.

## 📊 카테고리별 계산 방법

### 계산 우선순위
모든 카테고리는 다음 우선순위로 계산됩니다.

1. **Climatiq API 사용**
   - 전세계 산업 연관 분석 데이터(exiobase) 활용
   - Global region 우선 사용
   - API 실패 시 로컬 배출 계수로 자동 전환
2. **로컬 배출 계수 사용 (Fallback)**
   - IPCC 및 검증된 공개 데이터 기반

---

### 1. 교통 🚗
- **입력 단위**: 분 또는 km  
- **계산 방식**
  - 분 단위 입력 시 평균 속도로 km 변환 후 계산
    - 자동차: 30 km/h
    - 버스: 25 km/h
    - 지하철: 30 km/h
  - 걷기/자전거: 0 kgCO₂e

- **Fallback 배출계수 (kgCO₂e/km)**
  - 자동차: 0.171
  - 버스: 0.089
  - 지하철: 0.014

**예시**  
자동차 30분 이동 → 15km × 0.171 = **2.565 kgCO₂e**

---

### 2. 의류 👕
- **입력 단위**: 개
- **계산 방식**
  - Climatiq API 사용 시 평균 무게 환산
  - 빈티지 의류는 새제품의 10% 배출량 적용

- **Fallback 배출계수 (kgCO₂e/개)**
  - 티셔츠: 2.0 (빈티지 0.2)
  - 청바지: 33.4 (빈티지 3.34)
  - 신발: 13.6 (빈티지 1.36)

---

### 3. 식품 🍖
- **입력 단위**: 회(한끼 기준)
- **기준**: 한국일보 한끼 밥상 탄소 계산기
- **특이사항**
  - 파스타 항목만 Climatiq API 사용
  - 1회 = 0.25kg 환산 후 계산

**예시**
- 김밥 2회 → 0.8 kgCO₂e  
- 물냉면 1회 → 2.4 kgCO₂e

---

### 4. 쓰레기 🗑️
- **입력 단위**: kg 또는 개
- **개 → kg 변환**
  - 캔: 0.015 kg
  - 병: 0.4 kg

- **배출계수 (kgCO₂e/kg)**
  - 일반: 0.5
  - 플라스틱: 2.5
  - 종이: 0.3
  - 유리: 0.2
  - 캔: 1.5

---

### 5. 전기 ⚡
- **입력 단위**: 시간
- **소비전력**
  - 냉방기: 2.0 kW
  - 난방기: 1.5 kW
- **배출계수**: 0.424 kgCO₂e/kWh

---

### 6. 물 💧
- **입력 단위**: 회 또는 분
- **사용량 기준**
  - 샤워: 70 L/회
  - 설거지: 15 L/회
  - 세탁: 60 L/회
- **배출계수**: 0.0003 kgCO₂e/L

---

## 📐 단위 변환 규칙

| 항목 | 변환 기준 |
|---|---|
| 교통 | 분 → km (속도 기반) |
| 식품 | 1회 = 한끼 |
| 쓰레기 | 개 → kg |
| 전기 | 시간 × 소비전력 |
| 물 | 회/분 → L |

## 💰 포인트 지급 기준

### 1. 절약량 포인트
- 절약 탄소량(kg) × 100점

### 2. 빈티지 제품 포인트
- 제품 1개당 10점

### 3. 평균 대비 감축 포인트
- (평균 - 개인 배출량) × 20점 (최대 100점)

**예시**
- 자전거 5km + 빈티지 의류 1개 + 평균 이하 배출  
→ **총 144점**
